## 🚀 Project Overview  

This project implements an **autonomous self-healing cloud system** that:

- Monitors CPU, RAM, Disk in real-time  
- Detects failures automatically  
- Classifies system issues  
- Executes recovery actions  
- Logs events using blockchain-style structure  
- Displays everything in a live dashboard  

---
