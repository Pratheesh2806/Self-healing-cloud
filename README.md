## 🚀 Project Overview  

This project presents a **Decision-Driven Autonomous Self-Healing Cloud Infrastructure** designed to monitor, detect, analyze, and recover system failures **automatically without human intervention**.

It integrates:
- ⚡ Real-time monitoring  
- 🧠 Intelligent decision engine  
- 🔄 Autonomous recovery  
- 🔐 Blockchain-based logging  
- 📊 Advanced visualization dashboard  

---

## 🎯 Key Highlights  

✨ Fully automated failure detection and recovery  
✨ Real-time system monitoring (CPU, RAM, Disk)  
✨ AI-based analysis using **Neural Insight Engine**  
✨ Blockchain-style tamper-proof logs  
✨ Interactive dashboard with graphs and animations  
✨ Trust score evaluation system  
✨ Incident timeline and recovery tracking
